Python is a powerful programming language that is widely used across many fields and industries. Many employers are actively seeking programmers with experience in Python.

Python is an "open" language, and it can be freely downloaded and installed on almost any computer. One of its great advantages is that there are many freely available software libraries that can be downloaded and used that numerical methods (e.g., numpy), optimization (e.g., pyomo), machine learning (e.g., pytorch, scikit-learn, tensor-flow), thermodynamics and reaction engineering (e.g., Cantera), and data handling (e.g., Pandas). In addition, there is a large open source community that actively develop and contribute codes for others to freely use.

The beauty of Python is that once you learn some of its basic concepts, you can easily get started in using any of these software libraries to develop your own applications.

This module will give an introduction to Python, starting with using it as a simple calculator. Then, we will give more sophisticated uses. We will leave it to you to push your exploration and learning of Python to as far you want to go.

Previously, we mentioned that Python could be used as a calculator. This is true, however, if you work with plain Python, you will notice that there are several functions missing from it that are present in modern calculators, such as logarithms, trigonometric functions, hyperbolic trigonometric functions, etc. and special constants, like pi and Euler's number.

While these features are not in Python by default, we can import them from external libraries, which are collections of functions written by others. Some key libraries that we will be using throughout this module, and that you will find extremely useful, include:

  • Numpy
  • Matplotlib / Pylab
  • Scipy
  • Sympy
