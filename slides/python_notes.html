  • data for nitrogen

As an example, the vapor pressure of a fluid is often reported using the Antoine equation with a set of specific values for the coefficients which depend on the particular fluid. The value of these coefficients are typically only valid over a specific range of temperatures. 


def get_Cp(T):

    if (T >= 100.0 and T <= 500.0):
        A =  28.98641
        B =  1.853978
        C = -9.647459
        D =  16.63537
        E =  0.000117
    elif (T > 500.0 and T <= 2000.0):
        A =  19.50583
        B =  19.88705
        C = -8.598535
        D =  1.369784
        E =  0.527601
    elif (T > 2000.0 and T <= 6000.0):
        A =  35.51872
        B =  1.128728
        C = -0.196103
        D =  0.014662
        E = -4.553760
    else:
        print('error: Temperature out of range')
        exit()

    t = T / 1000.0
    Cp = A + B*t + C*t**2 + D*t**3 + E / t**2

    return Cp

print(get_Cp(5000.0))
