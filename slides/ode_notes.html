First order differential equations

Separable

\begin{align*} \frac{dy}{dx} &= f(y) g(x) \end{align*}

common mistakes include not fully isolating the terms (i.e.\ leaving a function of \(y\) on the $x$-side or a function of \(x\) on the $y$-side). The reason that this is incorrect is that we do not know a~priori how \(y\) depends on \(x\), or vice versa, and so we cannot perform any integral.

Linear

\begin{align*} y'(x) + p(x) y(x) &= f(x) \end{align*}

Integrating factor

integrating factor

\begin{align*} P(x) &= \int dx'\, p(x) \end{align*}
\begin{align*} e^{P(x)} y'(x) + e^{P(x)} p(x) y(x) &= f(x) e^{P(x)} \\ [e^{P(x)} y(x)]' &= f(x) e^{P(x)} \end{align*}

A couple of points… What if we include an integration constant in \(P(x)\) when determining the integrating factor? Let's see why this does not really matter. If we include this extra term, it will add a constant, multiplicative factor to the integrating factor. This can be canceled from all terms in the equation. So adding an constant to \(P(x)\) will not impact the solution.

We integrate this equation from a point \(a\), which is typically chosen where we know the function (i.e./ initial condition).

\begin{align*} \int_a^x dt\, [e^{P(t)} y(t)]' &= \int_a^x dt\, f(t) e^{P(t)} \\ e^{P(x)} y(x) - e^{P(a)} y(a) &= \int_a^x dt\, f(t) e^{P(t)} \\ y(x) &= e^{-P(x)+P(a)} y(a) + \int_a^x dt\, f(t) e^{-P(x)+P(t)} \end{align*}

Any linear first order differential equation can be solved analytically.
